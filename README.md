# Leasing-Bancario-Chile
El dataset proviene de la CMF (Comisión para el Mercado Financiero), regulador financiero de Chile. Contiene estadísticas mensuales del mercado de leasing bancario —arrendamiento financiero— desde enero 2013 hasta febrero 2026, cubriendo número de contratos, valor actual de cartera, tipo de bien, situación de mora e institución financiera.
